# Fundamentos de Programación en C

Este repositorio son prácticas de **Fundamentos de Programación en C** que he decidido guardar aquí porque resumen bastante bien cómo pienso cuando programo.

**Gestión de aficionados**

- Registra varios aficionados.
- Calcula una puntuación según años de membresía y antecedentes.
- Muestra quién queda primero aplicando un criterio de desempate.


**Estadio – entradas y gasto medio**

- Carga los datos desde un fichero de texto.
- Lista los aficionados.
- Permite elegir uno por id.
- Enseña todas sus entradas.
- Pide un tipo de evento y calcula el gasto medio de ese aficionado en ese tipo.
